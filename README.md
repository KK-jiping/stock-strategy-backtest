# 三井不動産 株価戦略バックテスト

## 概要
Pythonを用いて三井不動産（8801.T）の株価データを分析し、
移動平均クロス戦略にMACDフィルターを組み合わせた
アルゴリズム売買戦略のバックテストを実装しました。

## 使用技術
- Python
- yfinance（株価データ取得）
- pandas / numpy（データ分析）
- matplotlib / seaborn（可視化）

## 実装内容
1. 株価データの取得と可視化
2. 基準化比較と相関分析
3. 日次リターンの分布分析
4. 移動平均クロス戦略の実装
5. MACDフィルター付き非対称戦略の実装
6. バックテスト（シャープレシオ・最大ドローダウンの計算）

## 結果
| 戦略 | 総リターン | シャープレシオ |
|------|-----------|--------------|
| バイ＆ホールド | 58.45% | 0.451 |
| 移動平均クロス戦略 | 66.14% | 0.523 |

## 今後の改善予定
- オーバーフィッティングの検証
- 学習期間で最適化したパラメータはテスト期間で性能が劣化する
- MA5という敏感すぎるパラメータはノイズを拾いすぎる
- 単一銘柄・単一戦略では安定したリターンを出すことが困難
- 複数銘柄への拡張
- リスク指標の追加（カルマーレシオ等）
